# Mensagem de amostra para parceiros após o processamento de entrada{#sample-message-to-partners-after-inbound-processing}

Sempre que um [!UICONTROL Server-to-Server] arquivo de entrada é processado, um recibo é enviado por email para soluções de parceiros e, se configurado, para o parceiro.
