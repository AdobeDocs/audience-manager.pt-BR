# Implementing Audience Manager {#implementing-audience-manager}

This section outlines and explains the processes related to getting started with the Audience Manager data management platform ([!DNL DMP]). Esta seção foi projetada para ajudar equipes de negócios, gerentes de projeto e gerentes de tecnologia a compreenderem o processo de implementação do Audience Manager. A introdução ao Audience Manager pode levar aproximadamente seis semanas para três meses, dependendo das suas necessidades de coleta de dados.

Nossas técnicas de implementação ajudam a criar parceria consultiva com novos clientes. Esse processo foi projetado para:

* Discover e compreender os requisitos de negócios;
* Produza um plano acionável para atender a essas exigências;
* Desenvolver soluções personalizadas para ajudar a atender a requisitos exclusivos ou casos de uso;
* Certifique-se de que seus dados proprietários sejam importados e disponibilizados no Audience Manager.

As equipes de Parceiros e Gerenciamento de contas trabalharão muito com você antes, durante e depois do processo de implementação.

O Audience Manager adota uma abordagem padrão para configurar e implementar.

## Build, Test, and Train Phase {#build-test-train-phase}

Durante a criação, teste e fase de treinamento, você analisará a estratégia e o protótipo da coleção de dados com um cliente designado nas Soluções de parceiro.

<!-- c_build_test_train.xml -->

Your data collection strategy will undergo end-to-end [!DNL QA] testing. As soluções de parceiros rastrearão erros detectados e as resoluções de problemas de coordenadas com nossos engenheiros de sistemas. O treinamento do cliente pode começar em paralelo com esses outros esforços.

A tabela a seguir descreve as principais atividades que ocorrem durante esta fase:

| Atividade | Finalidade/Descrição | Participantes |
|---|---|---|
| Preparar uma estratégia de coleta de dados | Trabalhar com equipes técnicas da Adobe para criar um plano de coleta de dados que atenda às suas necessidades de negócios | Equipes comerciais e técnicas |
| Implantar e testar código | Teste a solução proposta em um ambiente de preparo e execute testes entre navegadores | Equipes técnicas |
| Verificar a funcionalidade e resolver erros | Examinar e comunicar resultados, resolver erros e retestar | Equipes técnicas |
| Treinamento do usuário | Fornecer educação e entender sobre recursos, ferramentas e relatórios do Audience Manager | Equipes de negócios |

O fornecimento para esta fase pode incluir:

* Um plano de coleta de dados concluído e aceito;
* [!DNL QA] Teste completo;
* Instruções básicas sobre os recursos da interface do usuário do Audience Manager;
* Aceitação e logon.

## Code Implementation {#code-implementation}

Though the deployment process may seem complex, the code implementation is as simple as adding a few lines of [!DNL JavaScript] adjacent to the closing `</body>` tag of your website.

<!-- c_code.xml -->

### Implantação

The Audience Manager code snippet calls [!DNL Akamai] to download the business rules set up previously in the user interface. Além disso, os navegadores cliente armazenam em cache essas informações, o que ajuda a reduzir os tempos de carregamento de página e servidor. Nosso código e metodologia de coleção de dados foi projetado para manter a experiência do usuário em seu inventário.

### Participantes

As Soluções de parceiros podem trabalhar diretamente com as equipes técnicas para ajudar a implantar o código, resolver problemas finais e atender a outros requisitos.

## Post-Implementation Support {#post-implementation-support}

Nossos esforços colaborativos não são interrompidos com a implantação final. Após concluir a implementação, a equipe de Gerenciamento de contas assume a tarefa.

<!-- c_post_implement_support.xml -->

Os gerentes de conta fornecem serviços de suporte e consultoria contínuos após o processo de implementação do produto ser concluído. Você pode contar com reuniões regulares com seu gerente de contas. Essas reuniões garantem que você obtenha o valor máximo de uso e valor do Audience Manager.
