      <li id="li_0B524C6767274E1BAADEE0833DA00C84"> <p>As expressões de limite de frequência incluem todos os usuários cujo número de realizações de característica está abaixo do valor desejado. </p> <p>Por exemplo: </p> <p> <code> frequency([1000T]) &lt;= 5</code> </p> <p>Essa expressão inclui todos os usuários que perceberam a característica com a ID "1000" menos de cinco vezes, incluindo usuários que não perceberam a característica. </p> </li>

      <li id="li_BAEC874C0B6E4232A83C7228E84A7AD0"> <p>Para anunciar casos de uso de limite de frequência, você pode criar uma regra de segmento semelhante ao seguinte: </p> <p> <code> (frequency([1000T] &lt;= 2D) &gt;= 5)?</code> </p> <p>Essa expressão inclui todos os usuários que perceberam a característica com a ID "1000" nos últimos 2 dias mais de cinco vezes. </p> <p>Você pode obter o limite enviando esse segmento para o servidor de publicidade e, em seguida, incluir um NOT no segmento no servidor de publicidade. Essa abordagem atinge maior desempenho no Audience Manager, ao mesmo tempo que atende o mesmo propósito para o limite de frequência. </p> </li>
